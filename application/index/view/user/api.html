接口说明

如果您有自己的网站，不管是（刷单平台、类似淘宝论坛、货源网站）等~~

 

只要你的网站是有淘宝或京东卖家的地方都是可以销售空包、流量、收藏、购物车、等本站的所有产品。 您只需要在您的网站开发一个可以让客户下单的功能然后对接我们的API就可以的了。

 

您客户购买以后直接传达到我们的网站，我们每天都会帮助您去发货的， 只要您那边对接好了以后所有功能都是自动的不需要人工进行任何的操作！

 

测试账号和密码就是您现在登录的账号和密码即可！

 

以下就是 API 对接地址：（提交方式是 POST 提交）


  • 获取快递价格地址（正式）：{$host_link}API/GetKd
  • 获取快递价格地址（测试）：{$host_link}TestAPI/GetKd
  • 获取流量价格地址（正式）：{$host_link}API/GetFlow
  • 获取流量价格地址（测试）：{$host_link}TestAPI/GetFlow
  • 获取收藏价格地址（正式）：{$host_link}API/GetSC
  • 获取收藏价格地址（测试）：{$host_link}TestAPI/GetSC
  • 快递单号下单地址（正式）：{$host_link}API/BuyKddh
  • 快递单号下单地址（测试）：{$host_link}TestAPI/BuyKddh
  • 流量收藏下单地址（正式）：{$host_link}API/BuyFlowSC
  • 流量收藏下单地址（测试）：{$host_link}TestAPI/BuyFlowSC
